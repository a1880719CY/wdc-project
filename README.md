# Template Repository for COMP SCI 2207/7207 Web & Database Computing (2023 Semester 1)

Contains environment files for WDC 2023. Copy this template for a general sandbox.

Auto commit/push/sync to Github is disabled by default in this template repository.
Enable the GitDoc extension to use this fucntionality (either in your VSCode settings, or in the Dev Container settings)

the sql version must be under 8.0 version to run the web successfully.

express-session must be installed using npm

googleapis must be installed using npm

the index.js must be modified depends on the users mysql like filling in the password etc, to make it work properly
